# Bank-Loan-Task2-EDA-SQL
Exploratory Data Analysis and SQL insights on bank loan dataset

---

## 📊 Objectives
- Analyze loan data using SQL
- Identify patterns and trends
- Generate business insights

---

## 🔍 Key Analysis Performed
- Total loan applications
- Monthly trends
- Good vs bad loan percentage
- State-wise loan distribution
- Purpose-wise funding
- Interest rate and DTI analysis

---

## 🛠️ Tools Used
- SQL Server
- Excel
- Tableau
- Jupiter Notebook

---

## 📈 Key Insights
- Majority loans are Fully Paid / Current
- Good loans account for 89.8%
- Highest loan applications from CA (California)
- Most common purpose is Debt consolidation

EDA and SQL analysis helped uncover important business insights that will be used for dashboard creation.
